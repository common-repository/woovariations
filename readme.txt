== Description ==

= ENG = 

**WooVariations** - Add opportunity to choose product variations on category/product list page with "Add to cart" button. Plugin work correct only with simple themes that don't overwrite the file **content-product.php**.

= RUS =

**WooVariations** - Добавляет возможность выбора вариаций (атрибутов) товара на странице каталога, а не только в карточке товара. Плагин будет правильно работать только с простыми темами, которые не перезаписывают файл **content-product.php**.

Author: [Flaeron](http://wp-lessons.com/o-avtore "Flaeron")

== Installation ==

**First download the ZIP file**
* Log in to your website administrator panel.
* Go to Plugins page, and add new plugin.
* Upload WooVariations.
* Click **Install Now** button.
* Then click **Activate Plugin** button.
